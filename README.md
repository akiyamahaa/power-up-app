# INSTALLATION
## Step 1: Clone my source code
- `git clone https://github.com/akiyamahaa/power-up-app.git`
## Step 2: Access the project directory
- `cd power-up-app`
## Step 3: Install dependencies
- `yarn install`
## Step 4: Run App
- `yarn start`
